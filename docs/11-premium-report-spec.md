## 3. 5 Metrics 시스템

### 3-1. Visual Weight Score (시각적 무게감)

```
정의: 이목구비가 얼굴에서 차지하는 시각적 비중
범위: 0-100
측정: 눈·코·입의 크기, 진하기, 농도감의 종합 점수

K-Beauty 활용:
  낮은 점수 (0-40): 담백, 연한 느낌 → "맑은 피부 + 자연 아이라인" 전략
  중간 점수 (41-70): 균형잡힌 인상 → 다양한 스타일 가능
  높은 점수 (71-100): 강렬한 인상 → 포인트 메이크업으로 밸런싱

Mini Card: 수치만 표시 (예: "VW: 42")
Premium: 수치 + 이유 + "시각적 무게를 높이려면 이 아이라이너 기법"
```

### 3-2. Canthal Tilt & Eye-Lift Index (눈꼬리 각도)

```
정의: 눈꼬리의 상승/하강 각도
범위: -10° ~ +15°
분류: Negative / Neutral / Positive

K-Beauty 활용:
  Negative (-10°~-2°): 부드러운 인상 → "puppy eyes" or 상승 아이라인
  Neutral (-2°~+5°): K-뷰티 골든 존 → 자연스러운 연장
  Positive (+5°~+15°): 더 카리스마 인상 → "cat eye" 강화 or 소프트닝

Mini Card: 각도 + 화살표 (예: "CT: +3° ↗")
Premium: 각도 분석 + 셀럽 대비 + "제니의 +8°에 가까워지려면..."
```

### 3-3. Mid-face Ratio & Philtrum Index (중안부 비율)

```
정의: 눈과 입 사이 거리 / 전체 얼굴 길이
범위: 비율 % + Philtrum 상대 길이
기준: 이상적 비율 33% (얼굴을 3등분했을 때)

K-Beauty 활용:
  비율 높음 (>36%): 성숙한 인상 → "동안 메이크업" (오버립, 언더 아이라인)
  비율 적정 (30-36%): 밸런스 → 스타일 자유도 높음
  비율 낮음 (<30%): 동안 → "어른스러운 메이크업" (매트 립, 가짜 아이브로우)

Mini Card: 비율 % (예: "MF: 37%")
Premium: 비율 + 동안/성숙 스펙트럼 위치 + "수지의 32%에 가까워지려면..."
```

### 3-4. Luminosity Score (K-Glass Skin 지수)

```
정의: 피부의 투명함·광택·텍스처 종합 점수
범위: 0-100
서브지표: 현재 점수 / K-Glow 적용 후 잠재 점수 / 텍스처 등급

K-Beauty 활용:
  "K-Glass Skin"은 K-뷰티의 아이콘
  이 점수가 높을수록 = "깨끗한 피부" 인상
  잠재 점수 = "이 제품들을 쓰면 여기까지 올라갈 수 있다" (구매 유도)

Mini Card: 현재 점수 + 잠재 점수 (예: "GLOW: 42 → 82")
Premium: 텍스처 분석 + 스킨케어 루틴 + 제품 추천
```

### 3-5. Harmony Index (조화 지수)

```
정의: 이목구비 비율·대칭·밸런스의 종합 조화도
범위: 0-100
서브지표: 전체 조화도 / 대칭 점수 / 최적 밸런스 포인트

K-Beauty 활용:
  "크니가 강조하는 '아름다운 것은 비율이다'"
  절대적 아름다움이 아닌, 비율의 조화를 측정
  메이크업으로 조화도를 높이는 방향 제시

Mini Card: 전체 점수 (예: "HARMONY: 68")
Premium: 비율 분석 + 어디가 불균형인지 + "이 부분을 보정하면 +20"
```
